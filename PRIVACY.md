# Privacy Policy

## Overview

This plugin provides LLM-based document reranking for Dify.
It sends reranking inputs to third-party model providers selected by the user.

## Data Processed

When reranking is invoked, the plugin may process:

- Query text
- Candidate document texts
- Model name and rerank parameters (such as window size and step size)
- Optional user identifier passed by Dify runtime

## Third-Party Processing

Depending on the configured model, data is sent to one of these providers:

- OpenAI
- Anthropic
- Google Gemini

Your use of these providers is subject to their own terms and privacy policies.

## Data Storage

This plugin does not intentionally persist rerank content (query/documents) in its own database.
It only processes data in memory during request handling.

## Credentials

API keys are provided through Dify credential settings.
The plugin uses these credentials only to call the selected model provider.

## Logs

The plugin does not intentionally log full document contents.
Operational/runtime logs may still be generated by the host environment (Dify, container platform, or infrastructure).

## Data Retention

This plugin does not define an independent retention system for rerank content.
Any retention behavior is determined by:

- The selected model provider
- Dify platform configuration
- Your deployment/infrastructure logging settings

## Security Responsibility

You are responsible for:

- Providing and managing API keys securely
- Configuring access controls in Dify
- Complying with applicable laws/regulations for the data you process

## Contact

For privacy-related concerns about this plugin, contact the plugin maintainer/publisher listed in the repository or marketplace metadata.
